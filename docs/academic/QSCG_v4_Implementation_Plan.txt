PHASE 1: AKADEMİK TEMEL (0-3 Ay)
================================

1.1 Kuantum Tünelleme Matematiği
--------------------------------
• Schrödinger Denklemi: iℏ∂Ψ/∂t = ĤΨ
• Tünelleme Olasılığı: T ≈ exp(-2/ℏ · ∫√[2m(V(x)-E)] dx)
• Heisenberg Belirsizlik İlkesi: Δx·Δp ≥ ℏ/2
• WKB Yaklaşımı (Wentzel-Kramers-Brillouin)

1.2 Lattice Kriptografi Teorisi
-------------------------------
• Module-LWE: b = A·s + e (mod q)
  - A: k×k matris over R_q = Z_q[x]/(x^n+1)
  - s: secret vector
  - e: hata vektör (discrete Gaussian)

• Module-SIS: A·z = 0 (mod q), ||z|| ≤ β
  - Short Integer Solution problem

• NTT (Number Theoretic Transform):
  - O(n log n) polynomial multiplication
  - In-place butterfly operations
  - AVX2/NEON vectorization

1.3 NIST Standartları
---------------------
• FIPS 203 (ML-KEM): Key Encapsulation Mechanism
  - ML-KEM-512 (Seviye 1)
  - ML-KEM-768 (Seviye 3) 
  - ML-KEM-1024 (Seviye 5)

• FIPS 204 (ML-DSA): Digital Signature Algorithm
  - ML-DSA-44 (Seviye 2)
  - ML-DSA-65 (Seviye 3)
  - ML-DSA-87 (Seviye 5)

• FIPS 205 (SLH-DSA): Stateless Hash-Based Digital Signature
  - SHA2-128s, SHA2-128f, SHA2-256s, SHA2-256f
  - SHAKE-128s, SHAKE-128f, SHAKE-256s, SHAKE-256f
