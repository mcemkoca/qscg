## Recommendations

### For Research / Education ✅
- Current performance is **acceptable** for learning and prototyping
- All operations complete in under 50 ms
- API design is stable and well-documented

### For Production ❌
- **Do not use** pure-Python QSCG in production
- Use optimized backends: [liboqs](https://github.com/open-quantumsafe/liboqs), [wolfSSL](https://www.wolfssl.com/), [BoringSSL](https://boringssl.googlesource.com/boringssl/)
- QSCG can serve as a **reference / test vector generator** for interop testing

### Optimization Path
1. **Numba JIT** for polynomial arithmetic (2-5× speedup expected)
2. **Cython** for NTT and matrix operations (5-10× speedup)
3. **Rust extension** via PyO3 (10-50× speedup)
4. **Full rewrite** in Rust with Python bindings (50-100× speedup)
