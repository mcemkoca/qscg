================================================================================
TROUBLESHOOTING
================================================================================

Problem: "fatal: Authentication failed"
Solution:
    git remote set-url origin https://USERNAME:TOKEN@github.com/mcemkoca/qscg.git
    # Or use SSH:
    git remote set-url origin git@github.com:mcemkoca/qscg.git

Problem: "fatal: refusing to merge unrelated histories"
Solution:
    git pull origin main --allow-unrelated-histories
    # Resolve conflicts, then commit and push

Problem: "error: failed to push some refs"
Solution:
    git pull origin main
    # Resolve conflicts
    git push origin main

Problem: Large file rejection
Solution:
    # If PNG files are too large, compress them:
    # Or use Git LFS:
    git lfs track "*.png"
    git add .gitattributes
    git commit -m "Add LFS tracking for images"
